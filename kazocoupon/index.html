お手伝い券をクーポンとして発行することができる Web アプリです。

Twitter の　#web1week という 1 週間で Web アプリを作成するイベントに参加した時に作成しました。

コロナ禍で休校になったこども達が、お家のお手伝いをはじめるキッカケになればいいなというコンセプトで作りました。 気軽に利用できるように、ログインなしで使うことができ、LINE で URL を送るだけでクーポンが発行できるようにしました。
